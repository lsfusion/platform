Оператор `CLOSE FORM` - создание [действия](../paradigm/Actions.md), которое закрывает открытый в [интерактивном представлении](../paradigm/In_an_interactive_view_SHOW_DIALOG.md) экземпляр формы с заданным идентификатором.

### Синтаксис

```
CLOSE FORM formId
```

### Описание

Оператор `CLOSE FORM` создаёт действие, которое закрывает у пользователя открытый экземпляр формы с идентификатором `formId` (в виде запроса клиенту). Идентификатор задаётся при открытии формы [оператором `SHOW`](SHOW_operator.md). Действие не имеет параметров и не использует [контекст](Action_operators.md#contextdependent). Если открыто несколько экземпляров с этим идентификатором, закрываются все они; если ни одного такого экземпляра нет, действие не имеет эффекта.

### Параметры

- `formId`

    [Строковый литерал](Literals.md#strliteral), идентифицирующий закрываемый экземпляр формы. Это тот же идентификатор, который задаётся при открытии формы оператором `SHOW`.

### Примеры

```lsf
FORM orders
    OBJECTS o = Order
    PROPERTIES(o) number, customer
;

// открытие формы с идентификатором экземпляра
openOrders ()  {
    SHOW 'ordersInstance' = orders NOWAIT;
}

// закрытие ранее открытого экземпляра по его идентификатору
closeOrders ()  {
    CLOSE FORM 'ordersInstance';
}
```
