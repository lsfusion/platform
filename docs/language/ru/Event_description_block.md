*Блок описания события* описывает [событие](../paradigm/Events.md) в различных инструкциях.

### Синтаксис

```
[GLOBAL | LOCAL] [FORMS formName1, ..., formNameN] [(GOAFTER | AFTER) propertyId1, ..., propertyIdM]
```

### Параметры

- `GLOBAL`

    Ключевое слово, при указании которого, описываемое событие будет глобальным. Является поведением по умолчанию.

- `LOCAL`

    Ключевое слово, при указании которого, описываемое событие будет локальным.

- `formName1, ..., formNameN`

    Список имен [форм](../paradigm/Forms.md), в которых будет происходить событие. Каждый элемент списка является [составным идентификатором](IDs.md#cid). Если список не задается, то событие будет происходить во всех формах.

- `GOAFTER` | `AFTER`

    Синонимичные ключевые слова; после любого из них следует список свойств или действий.

- `propertyId1, ..., propertyIdM`

    Список [идентификаторов](IDs.md#propertyid) свойств или действий. Этот список обозначает, что все обработчики событий, которые изменяют какое-либо из перечисленных свойств, а также сами перечисленные действия, должны выполняться раньше, чем обработчики, которые будут заданы в инструкции, для которой задается этот блок описания события.
