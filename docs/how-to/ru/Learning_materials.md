В дополнение к документации для знакомства с возможностями платформы **lsFusion** доступны следующие материалы.

### [Пособие для начинающих](https://lsfusion.org/ru/ed_materials)
Пособие по программированию на **lsFusion**, основанное на опыте автора. Книга даёт общее представление о процессе разработки прикладных решений, учит эффективно читать код, написанный на языке платформы, и приводит распространённые приёмы решения типовых задач, встречающихся в реальной разработке.

### [Видеоуроки](https://www.youtube.com/playlist?list=PL8FG-RmfUn1Oc30Vaw1Em3DeSzV5ElQ43)
Серия видео, демонстрирующих приёмы работы с платформой **lsFusion** в режиме реального времени.

### [Примеры](Examples.md)
Примеры разработки простых приложений, демонстрирующие возможности платформы.

### [Типовые задачи](How-to.md)
Примеры решения типовых задач, разбитые по категориям.
