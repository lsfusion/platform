Примеры предназначены для демонстрации возможностей платформы **lsFusion**.

### Турнирная таблица

Пример [Турнирная таблица](Score_table.md) описывает, как создать простое приложение для расчёта турнирной таблицы хоккейного турнира. В нём существует ровно одна форма, в которой пользователь может вводить результаты матчей, на основе которых автоматически строится турнирная таблица.

С помощью этого примера можно получить представление о том, как быстро разрабатывать приложения в стиле «Excel», в которых данные на формах доступны для редактирования и при любом их изменении все зависимые данные на форме обновляются инкрементно.

### Управление материальными потоками

Пример [Управление материальными потоками](Materials_management.md) описывает создание простого бизнес-приложения по управлению складом. В нём пользователь может фиксировать документы приходов и расходов, а также получать остатки по товарам.

В этом примере показан способ создания приложения по обработке документов, в которых есть шапка и строки. Все формы созданы по принципу «Dialog style». В этом подходе для каждого объекта в системе создаётся форма с их списком, в которой доступны для редактирования только кнопки по созданию, редактированию и удалению объектов. При нажатии соответствующей кнопки открывается отдельная диалоговая форма, при помощи которой пользователь добавляет новый или редактирует существующий объект.
