slug: "/Launch_events"
title: 'События запуска'
---

Событие запуска возникает при запуске одной из компонент платформы. Для каждого события запуска платформа предоставляет предопределенное действие, которое выполняется как обработчик этого события; разработчик встраивает в этот обработчик свою логику инициализации.

| Компонент         | Обработчик                            | Когда возникает событие                                                                                     |
| ----------------- | ------------------------------------- | ----------------------------------------------------------------------------------------------------------- |
| Сервер приложений | `SystemEvents.onStarted`              | Один раз при запуске сервера приложений, до того, как сервер начнет принимать подключения клиентов.        |
| Десктоп-клиент    | `SystemEvents.onDesktopClientStarted` | Один раз при запуске десктоп-клиента, после того, как клиент подключился к серверу приложений.             |
| Веб-клиент        | `SystemEvents.onWebClientStarted`     | Один раз при запуске веб-клиента, после того, как клиент подключился к серверу приложений.                 |
