Ручная установка предполагает отдельную установку компонентов платформы lsFusion поверх уже подготовленных дополнительных приложений (среда выполнения Java, PostgreSQL, сервер веб-приложений, среда разработки) — в том числе других версий и в окружениях, не покрываемых [автоматической установкой](Automatic_installation.md).
