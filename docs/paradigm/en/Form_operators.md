Form operators can be divided into several categories:

-   [Value input](Value_input.md) - operators responsible for inputting a value.
-   [Object group operators](Object_group_operators.md) - operators that operate on an [object group](Form_structure.md#objects) of a form.
-   [Focus operators](Focus_operators.md) - operators that manage the activity of form elements and report on it.
-   [Capture](Capture_SCREENSHOT.md) - an operator that creates an action capturing the currently rendered visual state of the user interface.
-   [Container visibility](Container_visibility_EXPAND_COLLAPSE.md) - operators that create actions controlling the visibility of the contents of collapsible containers of a form.
