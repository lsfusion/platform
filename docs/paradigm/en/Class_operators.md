Class operators include the following operators:

-   [Classification (`IS`/`AS`)](Classification_IS_AS.md) — determine whether an object belongs to a class.
-   [Signature matching (`ISCLASS`)](../language/Property_signature_ISCLASS.md) — determines whether a property can have a value for given objects, judged by the classes of those objects.
