---
title: 'Действия'
---

*Действие* - это элемент системы, который принимает на вход набор объектов (*параметров*) и с их использованием тем или иным способом изменяет состояние системы (как той в которой действие выполняется, так и состояние любой другой внешней системы).

Тип и поведение каждого действия определяется [оператором](Action_operators_paradigm.md), создающим это действие.

Действие может дополнительно объявлять *результат* — значение фиксированного класса, возвращаемое вызывающему коду через [оператор выхода](Exit_RETURN.md) внутри тела действия.

### Язык

Для добавления нового действия в систему используется [инструкция `ACTION`](ACTION_statement.md).

### Примеры

```lsf
CLASS Item;
inStock (Item i) = DATA BOOLEAN (Item);

// действие описывает эффект — здесь изменение хранимого свойства
markInStock (Item i)  {
    inStock(i) <- TRUE;
}

// действие может объявлять результат, возвращаемый вызывающему коду
getDescription (Item i) ABSTRACT STRING[100];
```
